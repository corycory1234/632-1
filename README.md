# 📊 銷售資料分析腳本

使用 Node.js 腳本讀取 JSON 格式的銷售交易資料，自動計算總收入、統計各商品售出數量，並輸出分析報告檔案。

## 📁 專案結構

```
project/
├── salesData.json   # 銷售原始資料
├── analyze.js       # 主分析腳本
└── report.txt       # 自動生成的分析報告
```

## ⚙️ 環境需求

- [Node.js](https://nodejs.org/) v18 以上版本

## 🚀 執行方式

**1. 克隆專案**

```bash
git clone https://github.com/你的帳號/你的儲存庫名稱.git
cd 你的儲存庫名稱
```

**2. 執行腳本**

```bash
node analyze.js
```

**3. 查看結果**

分析結果會同時顯示於終端機，並自動儲存至 `report.txt`。

## 📄 輸出範例

```
========== 銷售分析報告 ==========
分析交易筆數：10 筆
總收入：NT$ 1,925
---------- 各商品售出數量 ----------
西瓜：4 件
鳳梨：3 件
蘋果：4 件
芭樂：3 件
香蕉：4 件
---------- 最暢銷商品 ----------
西瓜、蘋果、香蕉：各售出 4 件
===================================
```

## 🛠️ 使用技術

- Node.js — 腳本執行環境
- `fs` 模組 — 檔案讀寫（Node.js 內建，無需安裝）
